### 物流大屏可视化项目


这是一个基于Vue3构建的数据可视化大屏项目，以智慧物流为背景构建出了一个智慧物流大屏展示项目

主要技术栈：

- vue3 + vite
- router + axios + moke
- vue-echarts(基本图表和地图引用)
- elementplus
